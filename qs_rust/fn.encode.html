Expand description

Encodes a Value tree into a query string.

The encoder preserves object insertion order by default, supports multiple list formats, and can be customized with filters, sorters, key/value encoders, and temporal serializers through EncodeOptions.

§Errors

Returns EncodeError when the supplied EncodeOptions are invalid or when encoding exceeds a configured maximum depth.

§Examples

use qs_rust::{EncodeOptions, Object, Value, encode};

let mut root = Object::new();
root.insert("a".to_owned(), Value::String("1".to_owned()));
root.insert("b".to_owned(), Value::String("2".to_owned()));

let encoded = encode(&Value::Object(root), &EncodeOptions::new()).unwrap();
assert_eq!(encoded, "a=1&b=2");
