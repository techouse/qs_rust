Struct DecodeOptions 

Source 
pub struct DecodeOptions { /* private fields */ }
Expand description

Options that control query-string decoding.

The defaults are chosen to match the common qs behavior: UTF-8 decoding, & as the delimiter, five levels of depth, list parsing enabled, and duplicate values combined into arrays.
